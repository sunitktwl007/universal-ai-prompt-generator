# 🚀 Universal AI Prompt Generator

A **fast, responsive web application** for generating optimized AI prompts. Perfect for creating tailored prompts for ChatGPT, Midjourney, Gemini, Claude, and more platforms.

![Version](https://img.shields.io/badge/version-1.0-blue) ![License](https://img.shields.io/badge/license-MIT-green) ![Status](https://img.shields.io/badge/status-active-brightgreen)

---

## ✨ Features

### 🎯 Intelligent Prompt Generation
- **5 Prompt Types**:
  - **Answer** - Structured responses with expert perspectives
  - **Image Generation** - Detailed visual prompts for AI artists
  - **Coding** - Production-ready code solutions
  - **Marketing** - Compelling sales copy & campaigns
  - **Video Script** - Scene descriptions with timing

- **4 Tone Options**: 
  - Professional - Formal, structured
  - Casual - Friendly, conversational
  - Aggressive - Bold, direct, high-impact
  - Creative - Innovative, artistic, unique

- **5 Platform Targets**: ChatGPT, Midjourney, Gemini, Claude, Generic
- **Smart Templates** - Optimized prompts based on your selections

### 🔊 Interactive Audio System
- **Web Audio API** - Smooth, professional sound effects
- **Click Sounds** - 600-1200Hz frequency sweep for interactions
- **Success Sounds** - C-E-G chord celebration
- **Sound Toggle** - Easy mute/unmute functionality
- **Graceful Degradation** - Works even if audio unavailable

### 💾 Smart Storage & Sharing
- **LocalStorage Integration** - Auto-saves up to 10 recent prompts
- **One-Click Copy** - Copy to clipboard with toast confirmation
- **Download Feature** - Export prompts as .txt files
- **Real-time Stats** - Word and character counting
- **Quick Access** - Recent prompts panel for fast reuse

### 🎨 Modern UI/UX Design
- **Dark Theme** - Eye-friendly gradient interface
- **Glassmorphism Panels** - Frosted glass effect with backdrop blur
- **Responsive Layout** - Optimized for desktop, tablet, and mobile
- **Smooth Animations** - Transitions, loading states, hover effects
- **Accessibility** - Keyboard shortcuts (Ctrl+Enter to generate)
- **Toast Notifications** - Non-intrusive user feedback

## 📖 How to Use

### Step-by-Step
1. **Enter Your Idea** - Describe what you need (up to 500 characters)
2. **Select Prompt Type** - Choose from Answer, Image, Coding, Marketing, Script
3. **Pick a Tone** - Professional, Casual, Aggressive, or Creative
4. **Choose Platform** - ChatGPT, Midjourney, Gemini, Claude, or Generic
5. **Click Generate** - Prompt is created instantly (~800ms)
6. **Use Your Prompt** - Copy to clipboard or download as text file

### Keyboard Shortcuts
| Shortcut | Action |
|----------|--------|
| `Ctrl+Enter` | Generate Prompt |
| Click Recent Item | Load previous prompt |

## 🔧 Troubleshooting

### Sound Not Working?
- Check browser volume settings
- Verify Web Audio API support
- Try with headphones
- Disable browser extensions

### Prompts Not Generating?
- Ensure JavaScript is enabled
- Check browser console for errors (F12)
- Clear browser cache
- Try different browser

### LocalStorage Not Saving?
- Disable private/incognito mode
- Check browser storage permissions
- Ensure sufficient disk space
- Try clearing old prompts manually
